.. _ref-gis-admin:

======================
GeoDjango's admin site
======================

.. module:: django.contrib.gis.admin
   :synopsis: GeoDjango's extensions to the admin site.


``GeoModelAdmin``
=================

.. class:: GeoModelAdmin

The ``GeoModelAdmin`` class accepts the same attributes as Geometry widgets.
Setting those attributes on the ``GeoModelAdmin`` class will automatically set
them on all the geographic widgets of the form. For a full list of valid
attributes, see :ref:`ref-gis-widgets-api`.

Additionally, a ``widget`` attribute is supported on the ``GeoModelAdmin``
class. Set it to an available base widget to override the base widget of all
the geographic fields of the admin form. For instance, to display 800*600
google maps on the admin form::

    from django.contrib.gis import admin, forms

    from myapp.models import SomeModel


    class SomeModelAdmin(admin.GeoModelAdmin):
        map_width = 800
        map_height = 600
        widget = forms.GMapWidget

    admin.site.register(SomeModel, SomeModelAdmin)

``OSMGeoAdmin``
===============

.. class:: OSMGeoAdmin

   A subclass of :class:`GeoModelAdmin` that uses a spherical mercator
   projection with `OpenStreetMap <http://openstreetmap.org/>`_ street data
   tiles. See the :ref:`OSMGeoAdmin introduction <osmgeoadmin-intro>` in the
   tutorial for a usage example.
